### 9.1 プロジェクト設定

サイドバーから「設定」メニューを選択すると、プロジェクト設定画面が表示されます。

**設定項目：**
- プロジェクト名
- プロジェクト説明
- その他のプロジェクト設定

### 9.2 ドメイン知識の管理

ドメイン知識は、AIがインタビューや業務生成を行う際の参照情報として使用されます。
プロジェクト固有の業務知識を登録することで、AIの出力精度を向上させることができます。

**ドメイン知識の追加：**

1. 設定画面で「ドメイン知識」タブを選択
2. 「新規追加」ボタンをクリック
3. 以下の情報を入力：
   - **業務**（必須）：ドメイン知識の名前
   - **知識**（必須）：具体的な業務知識の内容
4. 「保存」をクリック

**編集・削除：**
- 登録したドメイン知識は、一覧から選択して編集・削除が可能です

### 9.3 通知設定

OroSeeでは、システム内通知機能を使用してジョブの完了や重要なイベントをお知らせします。

**通知の種類：**

| レベル | 説明 |
|-------|------|
| **info** | 処理開始などの情報 |
| **success** | 処理完了の成功通知 |
| **warning** | 注意が必要な警告 |
| **error** | エラー発生の通知 |
| **requireAction** | ユーザーのアクションが必要 |

**通知の確認：**
- 画面右上の通知アイコンをクリック
- 未読通知がある場合はバッジで件数を表示
- 通知をクリックして詳細を確認
