description: 用語集
---


| 用語 | 説明 |
|-----|------|
| **インタビュイー** | インタビュー対象者 |
| **プロジェクト管理者** | プロジェクト管理者を作成できる、すべてのプロジェクトリソースの管理ができる役割 |
| **プロジェクト編集者** | プロジェクトにメンバーを追加できる、すべてのプロジェクトリソースの管理ができる役割 |
| **クライアント** | すべてのプロジェクトリソースの作成と編集ができる役割（削除は不可） |
| **閲覧者** | すべてのプロジェクトリソースの閲覧のみができる役割 |
| **ドメイン知識** | 特定の業界や業務領域に固有の専門知識 |
| **クイックウィン** | 短期間で成果が得られる自動化対象業務 |
| **ワークフロー** | 業務プロセスの流れを図式化したもの |
| **スイムレーン** | 担当者・役割別に分けたワークフロー表示形式 |
| **BPMN** | Business Process Model and Notation（業務プロセスモデル表記法） |
| **マジックリンク** | パスワード不要でログインできるリンク |
| **RPA** | Robotic Process Automation（業務プロセス自動化） |
