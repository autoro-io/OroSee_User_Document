**Q: ログイン用のメールが届きません**

A: 以下を確認してください：
1. 迷惑メールフォルダを確認
2. メールアドレスのスペルミスがないか確認
3. しばらく待ってから再度送信を試す

**Q: 招待リンクが無効と表示されます**

A: 招待リンクには有効期限があります。期限切れの場合は、プロジェクト管理者に再招待を依頼してください。

**Q: AI生成にはどのくらい時間がかかりますか？**

A: 処理するデータ量によりますが、通常以下の目安となります：
- 業務生成：30秒〜2分
- クイックウィン生成：1〜3分
- ワークフロー生成：1〜5分

処理完了は通知で確認できます。

**Q: 編集権限がないと表示されます**

A: プロジェクトでの役割（ロール）によって操作できる範囲が異なります。権限の変更が必要な場合は、プロジェクトのOwnerにご連絡ください。

**Q: データのバックアップはできますか？**

A: 現在、個別のバックアップ機能は提供していません。重要なデータは、業務一覧のコピー＆ペースト機能でExcelなどに保存することをお勧めします。
